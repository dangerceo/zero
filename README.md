# 0.computer

Control your Mac from your phone with AI-powered automation.

## Getting Started

For the easiest way to get started with zero, use the interactive onboarding flow:

```bash
npm run hello
```

This will guide you through the initial setup and get everything running smoothly.

### Standard Start

If you've already completed onboarding or want to run the standard dev server:

```bash
npm run dev
```

Then scan the QR code provided in the terminal from your phone to start using it!
